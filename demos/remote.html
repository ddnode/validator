示例：两个字段匹配

两个字段的值必须相同

JavaScript
$('#demo_21').validator ({
    fields: {
        'email': '邮箱:required;email',
        'againEmail': '确认邮箱: required;email;match[email]'
    }
});

两个字段比较大小

JavaScript
$('#demo_22').validator ({
fields: [{
    name: 'totle',
    rule: 'required;natural;match[gte,last]',
    display: '总量'
},{
    name: 'last',
    rule: 'required;natural;match[lte,totle]',
    display: '余量'
}]
});

示例：综合调用

自定义正则，自定义验证函数，自定义错误消息，其他参数

JavaScript
$('#demo_81').validator ({
fields: [{
    name: 'shopId',
    timely: false,
    rule: 'required',
    target: '#select_shop',
    msg: { rule: '请选择注册店铺!' }
},{
    name: 'cardNumber',
    rule: 'required',
    reg: /^\d{16}$/,
    msg: {reg: '请输入16位数字卡号'}
},{
    name: 'description',
    display: '描述',
    rule: 'required;safe',
    handler: function(el){
        if (el.value.indexOf('xxoo') !== -1) return '请文明用语!';
        return true;
    },
    success: function(el){
        $(el).css({color: '#000'});
    },
    error: function(el){
        $(el).css({color: '#f00'});
    }
}]
});

示例：动态更新规则

动态添加和移除字段规则（DOM上绑定的规则不会移除）

JavaScript
$('#demo_91').validator({
fields:[{
    name: 'cardNumber',
    rule: 'required'
}]
});
$('#btn_add_rule').on('click', function(){
$('#demo_91').validator('addField', {
    name: 'cardNumber',
    rule: 'required;number'
});
});
$('#btn_remove_rule').on('click', function(){
$('#demo_91').validator('removeField', {
    name: 'cardNumber'
});
});

示例：处理事件

表单验证成功事件以及字段验证失败事件

JavaScript
$('#demo_101').validator();
//表单验证成功事件
$('#demo_101').on('success.validate', function(){
$('#valid_msg101').text('success');
});
//字段验证失败事件
$('#demo_101').on('error.validate', 'input[name=cardNumber]', function(){
$('#valid_msg101').text('error');
});
